An application for configuring GitLab from an LDAP

This project maps the user configuration from an LDAP source to a GitLab server.

Many organizations use an LDAP server - or a system that exposes an LDAP interface - to define their users along with the users' access rights. If such an organization also uses GitLab to manage its Git source repositories, it may want to replicate the LDAP users configuration into its GitLab server. This is what gitSync implements. Running gitSync regularly - e.g., every hour or every day - maintains the equivalence of access rights between the two systems.
